## üíª Code Snippets & Examples

The following snippet demonstrates calculating the total revenue generated by each product, showcasing `JOIN`, `SUM()`, `GROUP BY`, and `ORDER BY`:

```sql
SELECT
    p.ProductName,
    SUM(od.Quantity * p.Price) AS TotalSales
FROM
    Order_Details AS od
JOIN
    Products AS p ON od.ProductID = p.ProductID
GROUP BY
    p.ProductName
ORDER BY
    TotalSales DESC;
```
